# PeopleIQ | HR Attrition Intelligence Platform

An advanced AI-powered HR analytics dashboard designed to predict and prevent employee attrition utilizing the IBM HR Attrition dataset.

![Dashboard Preview](https://via.placeholder.com/1200x600?text=PeopleIQ+HR+Attrition+Intelligence)

## 🚀 Features

- **3D Workforce Visualization**:Interactive Three.js tree representing organizational structure and department-level risk.
- **AI Strategic Insights**: Real-time recommendations powered by Large Language Models (integrated simulation).
- **Predictive Scenarios**: "What-If" simulator to model the impact of salary increases, overtime reduction, and satisfaction boosts on retention.
- **Risk Registry**: Individual employee-level risk scoring and monitoring.
- **Real-time Analytics**: Dynamic KPIs, attrition drivers, and department heatmaps.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (Rich Aesthetics, Glassmorphism)
- **Visualizations**: 
  - [Three.js](https://threejs.org/) (3D Org Chart)
  - [Chart.js](https://www.chartjs.org/) (Data Visualizations)
  - [GSAP](https://greensock.com/gsap/) (Smooth Animations)
- **Dataset**: IBM HR Attrition Dataset (N=1470)

## 📊 Dataset Insights

The underlying data model identifies **Overtime**, **Job Level**, and **Monthly Income** as the strongest predictors of attrition. The dashboard provides visual evidence for these correlations and allows HR professionals to model intervention strategies.
